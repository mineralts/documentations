# EmbedVideo

Represents the video of a MessageEmbed.

| Parameter | Type     | Description                 |
|-----------|----------|-----------------------------|
| url       | `?string` | URL of the video.           |
| proxy_url | `?string` | Proxied URL for this video. |
| height    | `number` | Height of the video.        |
| width     | `number` | Width of the video.         |
